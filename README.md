# Pokédex

A responsive Pokédex web application built with **HTML**, **CSS**, and **JavaScript**, using data from the PokéAPI.

## Features

* Search Pokémon by name
* Detailed Pokémon dialog with:

  * About
  * Base Stats
  * Evolution Chain
  * Moves
* Previous/Next navigation with keyboard arrow keys
* Loading animations for a better user experience
* Responsive design for desktop and mobile devices
* Data caching to reduce unnecessary API requests

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* PokéAPI

## API

This project uses the free PokéAPI:

https://pokeapi.co/

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/pokedex.git
```

2. Open the project folder.

3. Start the project by opening the `index.html` file in your browser.

No additional dependencies or build tools are required.
